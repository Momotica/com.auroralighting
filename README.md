# Aurora Lighting
This app adds support for Aurora Lighting AOne ZigBee devices in Homey.

## Supported Z-Wave devices with most common parameters:
AOne™ 120W Inline Dimmer (AU-A1ZB120)

## Supported Languages:
* English
* Dutch
